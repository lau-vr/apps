How to write a function:

  • Declare function parameters with f(x,y) = [parameters]
  • The first parameters are the variable units that the function depends on.
  • The last parameter is the actual function. (Example below)
  • Basic operations: +, -, *, /
  • Supports trigonometric functions ( sin, cos, tan ), powers ( ^ ), and factorials ( ! )
  • Order operations using: ' ( ' and ' ) '
  • Spaces are optional.
